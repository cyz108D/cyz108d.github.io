  4. 在本地建立一个CoolWeather文件目录

  6. CoolWeather下将会有一个coolWeather克隆下来的文件夹,把里面的东东复制到外层,删除原来的coolWeather文件夹

全代码流程(静态网页)

  1. 在本地建一个文件夹名字叫 USER.github.com
  2. 把你的静态文件放到一个文件夹，命个名，比如miao 。
  3. 然后把 miao 放进USER.github.com

  4. 接着打开 git bash （就是 git 的命令行）

  5. 有必要解释下上面这句，它的意思是在 远端 （remote）建立一个 git， 名字叫origin， 地址是后面那串 （其实这个解释是我猜的。。。）

把本地的内容 push 出去。
试试访问 USER.github.com 和 USER.github.com/miao/xxxx.html 吧~
