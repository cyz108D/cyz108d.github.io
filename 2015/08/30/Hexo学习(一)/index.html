搭建hexo博客

  1. 安装Git
    下载地址: http://git-scm.com/download/
  2. 安装Node.js
    下载地址: https://nodejs.org/
  3. 安装Hexo
    下载地址: https://hexo.io/

Hexo安装

这些命令是在git bash里进行的

命令1: npm install hexo-cli -g

输入这条命令时:出现了一个警告.
WARN optional dep failed,continuing fsevents

在网上搜索了下,好像是没什么影响.执行后:

两个文件夹里会生成一些必要的文件.

命令2: hexo init blog

这条命令,意思是在该条命令路径下,把hexo初始化内容拷贝到blog文件下
所以,其实可以不要命令中的blog.
hexo init直接把hexo初始化内容拷贝到当前命令的路径下.

命令3: cd blog

意思是进入blog文件夹,也就是hexo初始化内容的路径下.如果上条命令省略了blog,
就不用执行这条命令了,因为已经在hexo初始化内容的路径下了.

命令4: npm install

意思是安装hexo依赖包

命令5: hexo server

意思是启动本地hexo,执行后
出现了INFO Hexo is running……:4000Press CTRL+C to stop.
说明hexo已经启动了.
在浏览器里输入 http://localhost:4000/ 就会出现hexo网站的首页.

该条命令可以缩写为: hexo s

接下来必要命令

发布: hexo generate == hexo g
部署: hexo deploy == hexo d
这两条命令一般是同时进行,意思是有了新的文章,需要先发布,再部署到服务器上去,
这样网上的显示才有效果.
部署:在这里就是把发布的网站内容部署到github上去
